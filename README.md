# Hand Held Multi-Object Tracking Dataset in American Football

## Overview
We built Multi-Object Tracking Dataset in American football. 


## Dataset
- **AFMOT**  
  The first American football MOT dataset. [[ArXiv]](https://arxiv.org/abs/2511.09455)
